##### CommonJS

CommonJS规范概述了同步声明依赖的模块定义。这个规范主要用于在服务器端实现模块化代码组织，如果想要使用在浏览器端，需要转译下。

CommonJS模块定义需要使用require()指定依赖，exports对象定义公开的API。

```
module.exports = {
    name:'nick'
}


```
